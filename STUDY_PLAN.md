### Daily routine (keep it consistent)
- [ ] **Warm-up (10 min)**: re-solve yesterday’s easiest problem from memory (no peeking).
- [ ] **Core (60–120 min)**: solve today’s 3–6 problems.
- [ ] **Review (15 min)**: write 3 bullets: idea, invariants, complexity.
- [ ] **Exit criteria**: you can explain the approach + edge cases in < 2 minutes per problem.
