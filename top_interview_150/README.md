### How to run
Run any problem file directly; it will execute its built-in tests and then **exit**:
```bash
python top_interview_150/array_string/p0088_merge_sorted_array.py
```
### Notes
- **No duplicates**: if a problem already exists elsewhere in this repo, we link to that file instead of creating a new one.
- **Stubs**: newly generated files are stubs; implement the correct LeetCode method signature and add real asserts.
